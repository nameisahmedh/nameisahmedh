Searching for specific concepts (like "Naive Bayes") in 50+ hours of course footage is inefficient.

#### **The Innovation**
This RAG (Retrieval-Augmented Generation) system transcribes video data and uses semantic embeddings to provide answers grounded **strictly** in course content, eliminating hallucinations.
