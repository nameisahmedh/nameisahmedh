### ğŸ¥ **RAG-Based ML Teaching Assistant**
> **A semantic search engine for educational video content.**

<div align="center">
<a href="https://github.com/nameisahmedh/RAG-based-ML-teaching-Assistant">
  <img src="https://img.shields.io/badge/View_Source_on_GitHub-181717?style=for-the-badge&logo=github"/>
</a>
</div>

* **Problem:** Searching for specific concepts (e.g., "Naive Bayes") in 50+ hours of course footage is inefficient.
* **Solution:** Built a **Retrieval-Augmented Generation (RAG)** pipeline that transcribes video audio and uses semantic embeddings to find the exact video and timestamp.
* **Key Features:**
    * **Automated Transcription:** Integrated **OpenAI Whisper** for high-accuracy speech-to-text.
    * **Semantic Retrieval:** Uses **Vector Embeddings** to understand user intent beyond keywords.
    * **Precision Indexing:** Returns exact timestamps (e.g., *"Video 8, 0:31 - 1:05"*) for direct navigation.
* **Stack:** Python, Whisper, FAISS/ChromaDB (Vector Storage), LangChain, LLMs.
